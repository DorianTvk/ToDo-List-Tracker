# Task Management Application

A modern task management application built with TypeScript and React. Features include task organization by time periods (Today, Tomorrow, This Week), task completion tracking, and due date management.

## Tech Stack

### Frontend
- React with TypeScript
- Tailwind CSS for styling
- Shadcn UI components
- React Query for data fetching
- Framer Motion for animations

### Backend
- Express.js server
- PostgreSQL database with Drizzle ORM
- RESTful API endpoints

## Getting Started

1. Clone the repository
2. Install dependencies:
```npm install```
3. Start the development server:
```npm run dev```

The application will run on port 5000. Access it through your web browser at http://0.0.0.0:5000

## Features
- Create, update, and delete tasks
- Mark tasks as complete/incomplete
- Organize tasks by time periods
- Smooth animations and modern UI
- Responsive design
